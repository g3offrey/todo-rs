# todo-rs

A simple todo list written in Rust.

## Usage

### Help

```bash
todo-rs help
```

### Display TODOs

```bash
todo-rs ls
```

### Add TODO

```bash
todo-rs add "Buy milk"
```

### Complete TODO

```bash
todo-rs complete 1
```

or

```bash
todo-rs complete "milk"
```

### Delete TODO

```bash
todo-rs rm 1
```

or

```bash
todo-rs rm "milk"
```
